Finance - App para manejar finanzas personales

Una aplicación sencilla de finanzas que permite a los usuarios hacer un seguimiento de sus ingresos, gastos y presupuesto de manera fácil y eficiente. Proporciona una interfaz fácil de usar con funciones como agregar transacciones, categorizar gastos, generar informes y establecer objetivos.

PDF2Slide - Un lector de PDF liviano

Este es un lector de PDF que combina funcionalidad y estética, brindando a los usuarios una interfaz liviana y visualmente atractiva.
